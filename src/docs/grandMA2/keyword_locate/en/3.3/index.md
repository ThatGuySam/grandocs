		<p>To get the Locate keyword in the command line, type <strong>Locate </strong>or the shortcut <strong>Loca </strong>in the command line.</p>

<a name="toc_header_anchor_1" id="toc_header_anchor_1" class="topic-toc-item"></a><h2>Description</h2>

<p>The locate keyword is a function keyword to locate assigned objects on executors. The page will change to that page where the object is assigned and a HERE is displayed.</p>

<p>To turn locate off, repeat the Locate keyword with no object.</p>

<a name="toc_header_anchor_2" id="toc_header_anchor_2" class="topic-toc-item"></a><h2>Syntax</h2>

<p><span class="syntax">Locate [Object]</span></p>

<p><span class="syntax">Locate </span></p>
