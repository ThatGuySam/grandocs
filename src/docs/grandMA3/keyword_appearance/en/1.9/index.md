		<p>To enter the Appearance keyword in the command line, use one of the options:</p>

<ul>
	<li>Press <span class="hardkey">MA</span>&nbsp;+ <span class="hardkey">X4</span> + <span class="hardkey">X4</span></li>
	<li>Type <strong>Appearance</strong></li>
	<li>Type the shortcut <strong>Ap</strong></li>
</ul>

<a name="toc_header_anchor_1" id="toc_header_anchor_1" class="topic-toc-item"></a><h2>Description</h2>

<p>The Appearance keyword is used to assign an appearance to an object.&nbsp; &nbsp;</p>

<p>For more information see <a href="/Topic/7aa5737e-814e-454c-8468-23b912dfc010">Appearance</a>.&nbsp;</p>

<a name="toc_header_anchor_2" id="toc_header_anchor_2" class="topic-toc-item"></a><h2>Syntax</h2>

<p><span class="syntax"><strong>Assign Appearance ["Appearance_Name" or Appearance_Number] At [Object] ["Object_Name" or Object_Number]</strong></span></p>

<a name="toc_header_anchor_3" id="toc_header_anchor_3" class="topic-toc-item"></a><h2>Examples</h2>

<ul>
	<li>To assign appearance 14 to group 1, type:</li>
</ul>

<div class="cl_input">User name[Fixture]&gt; Assign Appearance 14 At Group 1</div>

<p>&nbsp;</p>

<ul>
	<li>To assign appearance "Sunset" to group 2, type:</li>
</ul>

<div class="cl_input">User name[Fixture]&gt; Assign Appearance "Sunset" At Group 2</div>

<p>&nbsp;</p>
