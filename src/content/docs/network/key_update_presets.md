You can modify the filter assignment for existing preset and update them.

**Requirement:** [Applied preset to a port](/network/key_assign_presets_to_ports/).

1. Click **Edit Presets** and then click **Extended filter and preset config**.
2. Modify the filter assignment for the preset in the assigned port column.
3. Click **Save as preset** at the end of the port column.\
   The save presets pop-up opens.\
   \
   ![](/img/network/save_preset_3_1-704e2d.png)\
   _Figure: Update Preset_\
    
4. Select in the drop down the preset you want to update.
5. Click **Update existing preset**.

The preset is updated with the new filter assignment and saved in the startup configuration.
