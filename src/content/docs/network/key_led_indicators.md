The following tables shows the LED indicators for the MA Network Switch along with the explanation of each indicator.

## Power LED

| Status    | Meaning         |
| --------- | --------------- |
| Off       | No power        |
| Green     | Power supply OK |
| Red blink | Error           |

## Status LED

| Status      | Meaning |
| ----------- | ------- |
| Green       | All OK  |
| Green blink | Update  |
| Red         | Error   |

## Link LED

| Status       | Meaning                |
| ------------ | ---------------------- |
| Off          | No link                |
| Green blink  | Gigabit connection     |
| Orange blink | 10/100 Mbps connection |

## Group LED

| Status  | Meaning |
| ------- | ------- |
| Red     | Group 1 |
| Yellow  | Group 2 |
| Green   | Group 3 |
| Cyan    | Group 4 |
| Blue    | Group 5 |
| Magenta | Group 6 |
| White   | Trunk   |
| Off     | Manage  |
