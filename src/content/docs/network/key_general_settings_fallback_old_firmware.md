You can downgrade the firmware of the MA Network Switch to the previous firmware version.\
The current firmware is displayed in the right corner at the bottom of the MA Network Switch display and in the web interface.

We recommend that the switches always run the same firmware in the same network.

![](/img/network/updatefirmware_3_1-795c5e.png)

**Requirement:** [Connection to the web interface](/network/key_access_web_interface/).

1. Click **Tools / Edit General Settings**.
2. Click **Update firmware**.
3. Click **Fallback to this firmware**.\
   A warning pop-up appears.

![](/img/network/popup_downdating-to-older-firmware_v3-3-6f1fdb.png)

_Downdating to older firmware_

4. Click **OK**.

The MA Network Switch starts to install the backup firmware and then reboots.
