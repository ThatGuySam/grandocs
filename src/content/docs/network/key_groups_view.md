The groups view is displayed after you have booted the MA Network Switch.

![](/img/network/img_groups-view_v3-4-dfc63a.png)

_Groups View_

The groups view displays the assigned group of each port in the group color.

The title bar displays the assigned switch name and the IP address.

If the background of a port starts to flash in red, the workload of the port is over 80 %.
