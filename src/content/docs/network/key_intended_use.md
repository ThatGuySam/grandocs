The MA Network Switch is intended to be operational right out of the box as a standard switch. In the default configuration, it will forward packets between connecting devices after power up.

Every other use is not intended and could follow personal injury and damage to property.

MA Lighting does not cover damages which can result of a not intended use of the console.
