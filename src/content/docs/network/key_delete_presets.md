If you created presets that you no longer use or need, delete them.\
You can delete previous created presets and the predefined presets.

**Warning:**\
After you delete a customized presets, you can not restore them.\
The predefined presets can be restored by a [factory reset](/network/key_reset_settings/).

**Requirement:** [Connection to the Web Interface](/network/key_access_web_interface/).

1. Click **Edit Presets**.
2. Click in the bottom right corner **Extended filter and preset config**.\
   \
   ![](/img/network/extended_mode_access_3_1-028143.png)\
   _Figure 1: Access Extended Mode_\
   \
   The extended mode opens.
3. Click in the bottom right corner **Delete presets**.\
   The delete presets pop-up opens.\
   \
   ![](/img/network/delete_presets_3_1-f3303b.png)\
   _Figure 2: Delete Presets Pop-Up_\
    
4. Select in the drop down the preset you would like to delete and click **Delete preset**.\
   A warning asks if you really want to delete this preset. Click **OK**.

The preset is deleted from the startup configuration. If the deleted preset was applied to a port, the filters in the preset are still applied to this port.
