The Effect key opens the Effects view. Here you can work with the different effects in the console.

To learn more about the effects and how to use them, please read the [What are Effects](/dot2/key_wa_effects/) and [How to Work with Effects](/dot2/key_ht_effects/).

## Advanced functionality

If you press and hold the ![ma](/img/dot2/ma-36da5c.png) key and then the Effects key, then you'll get the [SyncEffects command](/dot2/key_keyword_synceffects/).  This is used to synchronize multiple running effects.
