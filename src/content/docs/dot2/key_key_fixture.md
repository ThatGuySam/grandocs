The Fixture key will open the [Fixtures View](/dot2/key_viewitem_fixture/) on screen one. 

The key can also be used to directly access the [Fixture command](/dot2/key_keyword_fixture/), that can be used to select fixtures using the keys.

E.g. you want to select fixture 1 to 5 and 9. You can do this using these keys:

Fixture 1 Thru 5 + 9 Please

## Advanced Functionality

If you press and hold the ![ma](/img/dot2/ma-36da5c.png) key and then press the Fixture key gives you the [Selection Command](/dot2/key_keyword_selection/). This command doesn't do much. The only real purpose for this is in combination with Off. If you want to knockout the currently selected fixtures of the programmer, then you can do the following:

Off ![ma](/img/dot2/ma-36da5c.png)+Fixture Please

This will remove the programmer values of the selected fixtures and clear the selection.
