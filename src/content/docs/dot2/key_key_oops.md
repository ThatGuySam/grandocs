The Oops key performs an undo for most things you do in the console.

 E.g. if you have store the wrong cue, then you can press Oops and you'll no longer have the cue, but the values are back in your programmer.

 

If you are in the process of typing a command and haven't executed it yet (by pressing Please), then you can use Oops as a backspace function.
