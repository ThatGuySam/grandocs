To open the **Backup Window**, press Backup on the console.

![](/img/dot2/dot2_viewsandwindows_backupwindow01_1-2-a38b5c.png)

_Figure 1: Backup window_

In this window, you can create a new show, load a show, save the current show, and save the current show as.

Below the title bar is the name of the current show file along with the last save date and time displayed.

To create a new show, tap **New Show**. It opens the [New Show View](/dot2/key_window_newshow/).\
To load an existing show, tap **Load Show**. It opens the [Load Show View](/dot2/key_window_loadshow/).\
To save the current show, tap **Save Show**. It saves the show and close the **Backup Window**.\
To save the current show as a new filename, tap **Save Show as...** . It opens the [Save Show As View](/dot2/key_window_saveshowas/).

If a USB drive is attached, it is displayed below the four functions.\
If you save a show and a USB drive is attached, the show will be saved on the USB drive as well.

**Hint:**\
If a show was preprogramed in a previous version and saved, it can be loaded in a later version. If you save this show in a later version, you can no longer load it in a previous version. If you would like to keep using the show in a previous version, load the show and save it using a different name. 

Furthermore, you can enable the **auto save** option. By default, auto save is disabled.\
To select how often the console should auto save the show file, tap at the **three dots** ![](/img/dot2/dot2_viewsandwindows_addnewfixtureswindow01_1-0-b10b1a.png).\
You can choose between:\
\- every 15 minutes\
\- every 30 minutes\
\- every 60 minutes\
\- every 120 minutes

For more information see [how to save and load a show](/dot2/key_ht_saveandloadshow/).

To leave the **Backup Window**. Tap ![](/img/dot2/dot2_viewsandwindows_controlelements_titlebar08_1-0-5c7c42.png) in the [title bar](/dot2/key_title_bar/) or press Esc on the console.

## Encoder Bar Functions

![](/img/dot2/dot2_viewsandwindows_backupwindow02_1-2-b9c093.png)

_Figure 2: Encoder Bar Functions - Backup window_

**Auto Save:**\
To enable auto save and select how often the console should auto save, turn the encoder right.

**Select:**\
To select a function, turn the encoder left or right.\
To confirm a selected function, press or tap the encoder.
