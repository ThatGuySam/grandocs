The Go+ key is used to do a go forward (with time) on any executor. First press Go+ then any key associated with the executor.

Don't mistake this key with the big Go+ key under the main executor faders - Read about it [here](/dot2/key_key_defgoplus/).

As a default you already have the Go forward function on the ![go](/img/dot2/go_1-db7d7b.png) key in the executors with a fader. The executors without faders will have the [Toggle command](/dot2/key_keyword_toggle/) on the ![go](/img/dot2/go_1-db7d7b.png) key. You can change this functionality in the [Change Functions of Executor Buttons window](/dot2/key_window_changefunctionsofexecutor/).

## Advanced functionality

Pressing the ![ma](/img/dot2/ma-36da5c.png) key together with the Go+ key will give you access to the [Unpark command](/dot2/key_keyword_unpark/). This can be used to unpark (or release) any parked element.

E.g. you need to unpark fixture number 1:

![ma](/img/dot2/ma-36da5c.png) + Go+ Fixture 1 Please

Fixtures or attributes can be parked using the [Park command](/dot2/key_keyword_park/) or the Pause [key](/dot2/key_key_pause/).
