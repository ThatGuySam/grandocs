Pressing the Select key gives you the [Select command](/dot2/key_keyword_select/).

This can be used to select the fixtures used on an executor.

 

E.g. you need to select all the fixtures used in the cue list on executor 2:

Press Select and now press any of the keys associated with executor number 2.

 

Or if you need to select the fixtures stored in cue number 3 on executor 5:

Select Cue 3 Exec 5 Please
