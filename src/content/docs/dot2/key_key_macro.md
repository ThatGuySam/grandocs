When you press the Macro key, then you'll get the [macros pool view](/dot2/key_viewitem_macro/) on screen 1. Here you have direct access to all the macros.

 

If you want to directly run a macro you can also do this using the key.

E.g. if you want to run macro number one:

Macro 1 Please
