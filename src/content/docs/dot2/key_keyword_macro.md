This page describes the syntax and how to use the marco command.

To go to the macro command, press Macro on the console.

## Description

With the macro command, you start a macro from the [macro pool](/dot2/key_viewitem_macro/).

## Syntax

Start a macro.

Macro 1

## Example

Let´s assume, you will start macro 1 (e.g. +5 %) to the selected fixtures.

Press Macro 1 Please.

Macro 1 is started.
