The Clear key is used to release the selection and values in your programmer.

If you have some fixtures selected and you press the Clear key, then you will no longer have any fixtures selected.

If you have some values in your programmer but no selection, you will release the values and they will go back to the default values or values they might get from an executor.

If you have both a selection and values, you'll need to press the Clear key twice. First press will clear your selection, second clear will release the values. 

 

If you press and hold the Clear key for about a second, then you'll also clear both your selection and your values.
