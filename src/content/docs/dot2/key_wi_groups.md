A group is a way to store a selection of fixtures. 

If you often use the same selection of fixtures, then you might want to store them in a group. This allows you to easily call the same selection simply by selecting the group. 

A group also stores the selection order of your fixtures. A group with fixtures 1 + 2 + 3 is not the same as a group with fixtures 3 + 2 + 1.

You can also have groups with only one fixture.

Groups can be moved around in the groups view. This allows you to organize your groups (fixtures) in a way that makes sense for you.

They can be freely named so it's easy for you to remember what fixtures you have in the group.  

You can have several groups with the same selection.
