Pressing the If key gives you the [IfOutput command](/dot2/key_keyword_ifoutput/).

This can be used to select fixtures.

Depending on what you press after the If key you'll get a selection of different fixtures.

You can select fixtures based on what's currently on, currently using a preset, or is on and part of a specific group.

To learn more about the different options, please follow the link above.

## Advanced functionality

Pressing the ![ma](/img/dot2/ma_1-85eed0.png) together with the If key, will give you the [If command](/dot2/key_keyword_if/).

This allows you to limit selections of fixtures or as a limitation in various operations. Please follow the link above the learn more about the If command.
