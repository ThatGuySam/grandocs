The Go- key is used to execute a [GoBack command](/dot2/key_keyword_goback/) on an executor. It will take you to the previous cue with the cue timing. Press this key followed by any key associated with an executor.

Don't mistake this key with the big Go- key under the main executor faders - Read about it [here](/dot2/key_key_defgominus/).
