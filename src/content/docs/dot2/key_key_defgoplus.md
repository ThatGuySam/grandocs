The large     Go+     key under the main executor faders is locked to only perform a go forward with cue timing on the main cue list. The function can't be changed.

The command is actually DefGoForward.

Don't mistake this key with the small Go+ key in the command area - Read about that key [here](/dot2/key_key_goforward/).
