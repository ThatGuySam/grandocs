This page describes the syntax and how to use the Delay command.

To go to the Delay command it is necessary to press another function key before, e.g. Store and then three times Time.

## Description

With the Delay command you indicate delay times for cues.

## Syntax

Create a new cue 1 with a delay time of 3 seconds.

Store Cue 1 Delay 3

## Example

Let´s assume, you will store a new cue 1 on the main executor with a delay time of 3 seconds.

Press Store 1  (= Cue 1) three times Time (= Delay) 3 Please.

Cue 1 is stored on the main executor with a delay time of 3 seconds.

Double check the delay time in the [cues view](/dot2/key_viewitem_cue/).
