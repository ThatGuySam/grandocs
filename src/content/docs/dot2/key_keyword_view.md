This page describes the syntax and how to use the**** view command.

To go to the view command, press the ![view](/img/dot2/eye_1-352bc0.png) key on the console.

## Description

With the view command you can view the [cues of executors](/dot2/key_viewitem_cue/).

## Syntax

View the cues from executor 1.

View Executor 1

## Example

Let´s assume you will see the cues of executor 3 on page 1.

![](/img/dot2/dot2_commands_view01_1-2-f2198b.png)

_Figure 1: Cues View_

Press ![view](/img/dot2/eye_1-352bc0.png) and then respective executor button ![go](/img/dot2/go_1-db7d7b.png).

The cues view of the executor is visible on screen 1.
