The >>> key is used to go forward to the next cue without any cue timing and without running any automatic cue following.

Pressing >>> followed by a key associated with an executor will take you one cue forward, on that executor, in 0 seconds and will not run any timed cues or follow cue afterwards.

Pressing >>> Please will do this on the main cue list.
