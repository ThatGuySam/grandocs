1.  Install the grandMA3 I/O Node DIN-Rail onto the rail.

![](/img/grandma3/2-3/img_install_on_rail_1_1-6701bd.png)

Place the grandMA3 I/O Node DIN-Rail on the rail. 

 

![](/img/grandma3/2-3/img_install_on_rail_2_1-02228e.png)

Snap into place

2. Build the I/O Node DIN-Rail into the switchboard.

 

### Connections

![](/img/grandma3/2-3/img_gm3_io-node-din-rail_connections_power-661d0b.png)

Connect with electrical grid

-or- 

![](/img/grandma3/2-3/img_gm3_io-node-din-rail_connections_poe-1b7535.png)

Connect via PoE
