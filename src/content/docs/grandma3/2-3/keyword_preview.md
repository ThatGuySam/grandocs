To enter the Preview keyword in the command line, use one of the options:

- Press Preview
- Type **Preview**
- Type the shortcut **Previe**

## Description

The Preview keyword is used to toggle the preview mode. For more information on preview see [What is the Programmer](/grandma3/2-3/operate_programmer/). 

## Syntax

Preview (On/Off)

## Example

- To toggle the preview mode, type:

|                                                                    |                              |
| ------------------------------------------------------------------ | ---------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Preview  |

The title bar and the frame around the windows that display the preview mode turn red.
