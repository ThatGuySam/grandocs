The Show Creator menu lets you **Import** and **Export** various object types, **Create Presets**, **Create Groups**, **Store Presets to Fixture Types, and ****Create Presets from Fixture Types****.**

To learn more about **Import** and **Export**, see [Import Keyword](/grandma3/2-3/keyword_import/) and [Export Keyword](/grandma3/2-3/keyword_export/).

To learn more about creating various object types, for example, **Groups**, see [AutoCreate Keyword](/grandma3/2-3/keyword_autocreate/).

To learn more about how to **Store Presets to Fixture Types**, see [AutoStore Keyword](/grandma3/2-3/keyword_autostore/).
