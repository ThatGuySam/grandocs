To enter the FastSync keyword in the command line, use one of the options:

- Type **FastSync**
- Type the shortcut **Fas**

## Description

The FastSync keyword is a playback keyword that is used to determine the starting point of the phaser. The first step of the phaser is immediately started.

For more information on how to assign executors see [Assign Objects to an Executor](/grandma3/2-3/executor_assign/). 

## Syntax

**FastSync \[Object] \["Object\_Name" or Object\_Number]**

## Example

- To set the starting point of the phaser running in the sequence that is assigned to executor 128, type:
