To enter the Sound keyword in the command line, use one of the options:

- Type **Sound**
- Type the shortcut **Sou**

## Description

The Sound keyword addresses sound objects in the sound pool.

## Syntax

**\[Function] Sound \["Sound\_Name" or Sound\_Number]**

****

## Example

- To assign sound 5 to executor 401 on page 1, type:
