Pressing Freeze toggles the Freeze function.

|      |        |
| ---- | ------ |
| OK : | Freeze |

For more information about Freeze, see the [Freeze keyword](/grandma3/2-3/keyword_freeze/).

## Location

Freeze is located in the command section.

_Location on grandMA3 full-size and grandMA3 light consoles​_
