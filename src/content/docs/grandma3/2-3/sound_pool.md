The **Sounds** pool can contain sound files. The pool is part of the media pools.

|                                                  |                                                                                                                                                                                                        |
| ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ![](/img/grandma3/2-3/important_gray-f4dfbc.png) | **Important:**                                                                                                                                                                                         |
|                                                  | The Sounds pool has a limit of 100 MB for all objects. Files exceeding the limit will be highlighted with a red color in the import pop-up.The overall size of the media pools is a maximum of 200 MB. |

Sound pool objects can be played back in the pool or added as a track in a timecode object.
