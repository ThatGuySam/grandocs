To enter the **MemoryInfo **keyword in the command line, use one of the options:

- Type **MemoryInfo**
- Type the shortcut **Me**

## Description

The MemoryInfo keyword is a command keyword that is used to indicate how much memory is occupied. 

## Syntax

**MemoryInfo**

## Option Keywords

The MemoryInfo keyword uses the following option keywords:

- [/Limit](/grandma3/2-3/ok_limit/)
- [/Sort](/grandma3/2-3/ok_sort/)
