To enter the Speed keyword in the command line, type **Speed.**

## Description

The Speed keyword is used to set the speed of phasers.  

For more information see [Phasers](/grandma3/2-3/phaser/). 

**Requirement:**\
To set values to speed, there must be at least 2 steps in the programmer.

## Syntax

\[At] Speed \[Value]

## Examples

- To set the selected layer to speed, type:

|                                                                    |                            |
| ------------------------------------------------------------------ | -------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Speed  |

 

- To set the speed of the dimmer to 70 hertz, type:

|                                                                    |                                     |
| ------------------------------------------------------------------ | ----------------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>At Speed Hz 70  |
