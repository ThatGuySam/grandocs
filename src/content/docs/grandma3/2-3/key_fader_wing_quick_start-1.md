### Das Gerät anschließen und einschalten

1. Gerät auspacken.
2. Verpackungs- und Polstermaterial entfernen. 
3. Gerät in einem geschlossenen Raum auf eine stabile Fläche stellen. 
4. grandMA3 onPC Software auf www\.malighting.com herunterladen. 
5. Die Software auf PC oder Laptop installieren. Für Mindestanforderungen, siehe System Requirements im grandMA3 Online Benutzerhandbuch. 
6. Den grandMA3 onPC fader wing über das USB-Kabel mit dem PC oder Laptop oder einem command wing XT verbinden.
7. Den command wing XT oder den PC oder Laptop einschalten.
8. Den grandMA3 onPC fader wing einschalten:\
   -IEC-Kaltgerätestecker mit der dafür vorgesehenen Buchse verbinden.  \
   -Netzstecker an die Steckdose anschließen. \
   -Netzschalter auf **I **stellen.  \
   -Netzschalter leuchtet rot.

### grandMA3 onPC fader wing ausschalten

Um den grandMA3 onPC fader wing auszuschalten, Netzschalter auf **O **stellen.
