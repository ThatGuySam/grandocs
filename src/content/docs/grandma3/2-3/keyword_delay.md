To enter the Delay keyword in the command line, use one of the options:

- Press Time Time (If the Time Key Target is set to Fixture. For more information see [User Settings](/grandma3/2-3/user_settings/) and [Time Key](/grandma3/2-3/key_time/).)
- Press MA Time Time Time
- Type **Delay**
- Type the shortcut **Dela**

## Description

Delay sets the delay time of an object.

For more information on the delay times for cues see [CueDelay Keyword](/grandma3/2-3/keyword_cuedelay/).

## Syntax

Delay \[Delay\_Time]

## Example

- To set an individual delay time of 4 seconds in the current selection, type:

|                                                                    |                              |
| ------------------------------------------------------------------ | ---------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]> Delay 4 |
