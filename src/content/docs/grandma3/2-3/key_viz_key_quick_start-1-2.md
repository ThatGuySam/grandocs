### Mise en service

1. Déballer l'appareil.
2. Elever le matériel de protection. 
3. Brancher la viz-key grandMA3 à votre PC ou laptop utilisant un cable USB.
4. Pour éteindre la viz-key grandMA3, débranchez le connecteur USB.
5. Pour mettre à jour un appareil grandMA3, téléchargez le logiciel grandMA3 sur [www.malighting.com](https://www.malighting.com/).\
   Pour plus d'information sur Software Update, veuillez lire [Update grandMA3 viz-key](/grandma3/2-3/update_viz_key/).
