To enter the Station keyword in the command line, use one of the options:

- Type **Station**
- Type the shortcut **Stat**

## Description

The Station keyword is an object keyword which is used to address all stations in the network. You can also [invite](/grandma3/2-3/keyword_invite/) stations to your session or [dismiss](/grandma3/2-3/keyword_dismiss/) them.



## Syntax

\[Function] Station \["DeviceType\_Name" or DeviceType\_Number].\["Device\_Name" or Device\_Number]
