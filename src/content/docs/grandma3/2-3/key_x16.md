X16 | Exec is executor 198.

|                                            |                                   |
| ------------------------------------------ | --------------------------------- |
| ![](/img/grandma3/2-3/tip_gray-269535.png) | **Hint:**                         |
|                                            | All Xkeys behave like executors.  |

Executor

Pressing and holding MA + X16 | Exec enters the Executor keyword into the command line.

|                                                                    |                              |
| ------------------------------------------------------------------ | ---------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Executor |

​For more information about Executor, see the [Executor keyword](/grandma3/2-3/keyword_executor/).

SpecialExecutor

Pressing and holding MA + X16 | Exec + X16 | Exec enters the SpecialExecutor keyword into the command line.

|                                                                    |                                     |
| ------------------------------------------------------------------ | ----------------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>SpecialExecutor |
