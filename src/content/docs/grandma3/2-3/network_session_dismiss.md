Stations that are part of a session can be dismissed by another station.

**Requirement:**

A session needs to be running, and the station being operated needs to be part of the session. The station to be dismissed also needs to be in the same session. See how to create sessions in the [Create a Session topic](/grandma3/2-3/network_session_create/).

## Using the User Interface

Stations are located and dismissed using the [Network menu](/grandma3/2-3/network_session/).

1. Navigate to the Network menu - see how using the link above.
2. Locate and tap the station in the list of stations that are connected to the session (stations with a green background color).
3. Tap Dismiss Station at the bottom of the Network menu.

The selected station is now kicked out of the session.

## Using the Command Line

Stations are dismissed using the [Dismiss keyword](/grandma3/2-3/keyword_dismiss/).
