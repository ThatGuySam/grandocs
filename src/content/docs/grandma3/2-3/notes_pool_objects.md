The following example shows how to add a note to a macro. For more information about macros, refer to the [Macros topic](/grandma3/2-3/macros/).

- **Requirement**: The grandMA3 demo show is loaded. 

For an overview of the example below, have a look at this video:

[Vimeo video](https://player.vimeo.com/video/1026460754?title=0\&byline=0\&portrait=0\&color=ffeb0f)

To add a note to a macro, follow these steps:

1. Open the Add Window dialog, tap the Data Pools and then tap Macros. The Macro Pool window opens.
2. Press Edit and then tap the first macro pool object called Select EncoderBar. The macro editor opens.
3. Enable Settings in the title bar.
