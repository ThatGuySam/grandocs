### Start-up

1. Unpack the device.
2. Remove the packaging and cushioning material. 
3. Connect the grandMA3 viz-key via one of the USB cables with a PC or Laptop with installed grandMA3 onPC software.
4. To turn off the grandMA3 viz-key, disconnect the USB connector.
5. To update a grandMA3 device, download the latest software version from www\.malighting.com.\
   For more information, read the topic [Connect grandMA3 viz-key](/grandma3/2-3/fs_connect_viz_key/) in the online manual.
