### grandMA3  I/O Node

| -------------------------------- | --------------------------------------------------------- |
| Parameter                        | Keine                                                     |

| Gewicht (ohne Verpackung)        | 2 kg / 5 lbs                                              |

| Maße                             | 207 x 184 x 43 mm  / 9 x 8 x 2 in (Breite x Tiefe x Höhe) |
