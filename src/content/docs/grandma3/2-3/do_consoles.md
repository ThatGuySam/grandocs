The grandMA3 consoles control all kinds of lighting genres such as conventional lights, moving lights, LEDs, video, and media via DMX signal or within a network environment.

All grandMA3 components, despite different hardware solutions, use the same software. For a general overview of grandMA3 hardware, see  <https://www.malighting.com/grandma3/products>.

All components are fully integrable into the network environment.

This topic describes the different sections of the grandMA3 consoles.

These sections vary from model to model.

The flagship of the grandMA3 range is the full-size or full-size CRV, respectively.

The control room version (CRV) of full-size and light are models without the monitor wing, but come with additional external monitor ports.

The grandMA3 light console or the light CRV are the work-horses of the range.

The compact and compact XT models offer the full system benefits in a compact and lightweight format.

The grandMA3 consoles number their screens in a specific order, for more information see [Screen Allocation](/grandma3/2-3/do_screen_allocation/).

You can connect several external touch screens with a console, see [Connect External Screens](/grandma3/2-3/fs_connect_external_screens/).

The display elements of the screens can be modified, see [Configuration of Displays](/grandma3/2-3/ws_ui_display_configuration/).

 

For technical specifications, see [Technical Data](/grandma3/2-3/key_consoles_technical_data/) in the [grandMA3 Quick Manual consoles](/grandma3/2-3/grandma3_quick_manual_consoles/).
