To enter the Help keyword in the command line, use one of the options:

- Press Help
- Type **Help**
- Type the shortcut **H**

## Description

Help is a function keyword which is used to open the Help pop-up for you to search for topics or to use the context sensitive help.

## Syntax

Help \["Search\_Term"]
