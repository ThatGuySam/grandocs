To enter the ColorTheme keyword in the command line, use one of the options:

- Type **ColorTheme**
- Type the shortcut **Col**

## Description

ColorTheme is an object keyword which is used, for example, to adjust the color theme.

## Syntax

\[Function] ColorTheme
