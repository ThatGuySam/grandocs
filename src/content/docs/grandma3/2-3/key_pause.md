Pause

Pressing Pause Pause enters the Park keyword into the command line.

Fix

Pressing and holding MA + Pause enters the Fix keyword into the command line.

|                                                                    |                         |
| ------------------------------------------------------------------ | ----------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Fix |

​For more information about Fix, see the [Fix keyword](/grandma3/2-3/keyword_fix/).

Desk Lock

Pressing and holding MA + MA + Pause toggles the desk lock.

​For more information about the desk lock, see [Desk Lock](/grandma3/2-3/ws_ui_desk_lock/).
