The Network Interface menu lists the available interfaces, and in the grandMA3 hardware, it can be used to change the settings.

## Open the Network Interface Menu

1. Press Menu.\
   \- Opens the [menu select pop-up](/grandma3/2-3/wvm_menus/).
2. Tap Network.\
   \- Opens the Network menu.
3. Tap My Interfaces.\
   \- Opens the Network Interfaces pop-up:

Click this button to open the pop-up. [![](/img/grandma3/2-3/robot-icon_white-274940.png)       Paste to Command Line](#ma_cmd?Menu%20%27RemoteIpEditor%27)
