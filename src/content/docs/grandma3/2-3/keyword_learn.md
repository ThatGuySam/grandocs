To enter the Learn keyword in the command line, use one of the options:

- Type **Learn**
- Type the shortcut **Lear**

## Description

The Learn keyword is a playback keyword that is used to determine the starting point and the speed of a phaser at the same time. It is a combination of [LearnSpeed](/grandma3/2-3/keyword_learnspeed/) and [ReSync](/grandma3/2-3/keyword_resync/) keywords. 

For more information on the function Learn and phasers see [Phasers](/grandma3/2-3/phaser/).           

## Syntax

**Learn(\[Object] \["Object\_Name" or Object\_Number])**
