Pressing U2 executes the User2 command in the command line.

|                                                                    |                           |
| ------------------------------------------------------------------ | ------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>User2 |

This key is only present on grandMA3 compact, grandMA3 compact XT, grandMA3 onPC command wing and grandMA3 onPC command wing XT.

For more information about U2, see the [User2 keyword](/grandma3/2-3/keyword_user2/).

## Location

U2 is located in the command section on the left side of the numeric keys.

![](/img/grandma3/2-3/buttons_wings_u2_v0-99-e6d2b4.png)

_Location on grandMA3 compact consoles and grandMA3 onPC command wings_
