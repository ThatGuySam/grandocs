Pressing Highlt toggles the highlight function.

For more information about Highlight, see the [Highlight​ keyword](/grandma3/2-3/keyword_highlight/).

Pressing MA + Highlt toggles the lowlight function. For more information about Lowlight, see the [Lowlight​ keyword](/grandma3/2-3/keyword_lowlight/).

## Location

Highlt is located in the command section.
