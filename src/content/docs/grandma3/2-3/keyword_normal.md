To enter the **Normal **keyword in the command line, use one of the options:

- Press At At 
- Type **Normal**
- Type the shortcut **No**

## Description

The Normal keyword is an object keyword which is used to set the dimmer of the fixtures to the defined normal value.

For information on how to change th enormal value see [User Settings](/grandma3/2-3/user_settings/).

## Syntax

Normal

## Example

- To set the dimmer of the selected fixtures to the normal value, type:
