For more information about Off, see the [Off keyword](/grandma3/2-3/keyword_off/).

Off Menu

Pressing Off Off opens the Off menu.

Read more about the Off menu [here](/grandma3/2-3/executor_running_playbacks/).

Close RemoteHID Connection

Pressing MA + MA + Off closes every RemoteHID connection that is active.

For more information about RemoteHID connections, see the [RemoteHID](/grandma3/2-3/comad_remotehid/).

## Location
