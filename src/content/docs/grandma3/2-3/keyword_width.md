To enter the Width keyword in the command line, use one of the options:

- Type **Width**
- Type the shortcut **Wi**

## Description

The Width keyword is used to determine the width incorporating transition of the entire step. 

For more information see [Phasers](/grandma3/2-3/phaser/). 

## Syntax

Width

\[At] Width \[Value]

## Examples

- To set the layer to width, type:

|                                                                    |                            |
| ------------------------------------------------------------------ | -------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Width  |

 

- To create a PWM effect, type:

|                                                                    |                                                 |
| ------------------------------------------------------------------ | ----------------------------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>At Transition 0 + Width 50  |
