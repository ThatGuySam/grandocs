Layout

Pressing and holding MA + X4 | Layout enters the Layout keyword into the command line.

|                                                                    |                            |
| ------------------------------------------------------------------ | -------------------------- |
| ![](/img/grandma3/2-3/icon_commandline-input_logo_v2-0-59d23e.png) | User name\[Fixture]>Layout |
