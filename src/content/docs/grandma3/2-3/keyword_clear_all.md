To enter the ClearAll keyword in the command line, use one of the options:

- Type **ClearAll **
- Type the shortcut **Cleara**

## Description

The ClearAll function clears the selection and discards all values in the programmer.

For information on the key see the [Clear Key](/grandma3/2-3/key_clear/).

For more information on the additional functions of the Clear keyword see [Clear](/grandma3/2-3/keyword_clear/), [ClearActive](/grandma3/2-3/keyword_clear_active/), [ClearSelection](https://help.malighting.com/grandMA3/2.3/HTML/4c76b40d-6276-44d0-9e78-2dd9eed14a40.html).

## Syntax

ClearAll
