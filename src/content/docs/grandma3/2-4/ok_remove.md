To enter the **/Remove **option keyword in the command line, use one of the options:

- Type **/Remove**
- Type the shortcut** /R**

## Description

The /Remove option keyword is used to remove values that are stored in attributes and active programmer values. 

## Syntax

\[Function] \["Object\_Name" or Object\_Number] /Remove

## General Keywords

General keywords that use the /Remove option keyword:

- [Assign keyword](/grandma3/2-4/keyword_assign/)
- [Cook keyword](/grandma3/2-4/keyword_cook/)
- [CopyCrashLog](/grandma3/2-4/keyword_copycrashlog/)
- [Cue keyword](/grandma3/2-4/keyword_cue/)
- [Group keyword](/grandma3/2-4/keyword_group/)
- [Layout keyword](/grandma3/2-4/keyword_layout/)
- [MAtricks keyword](/grandma3/2-4/keyword_matricks/)
- [Preset keyword](/grandma3/2-4/keyword_preset/)
- [Store keyword](/grandma3/2-4/keyword_store/)
- [Update keyword](/grandma3/2-4/keyword_update/)
- [World keyword](/grandma3/2-4/keyword_world/)
