Cloning can be used for the following occasions:

- To clone data in the patch from existing fixtures to new fixtures, for example, if an additional truss is to be used in a rig on a stage.
- To transfer attribute values from one fixture to another in the programmer.
- To transfer values from one attribute to another attribute, for example, from Gobo1 to Gobo2.
- To transfer values from fixtures to fixtures throughout the show file, including Sequences, Groups, Presets, Worlds, and Layouts. 

Cloning can be triggered via the command line or a clone dialog in the user interface. For more information about the command line syntax, see [Clone Keyword](/grandma3/2-4/keyword_clone/).
