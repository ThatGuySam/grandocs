To enter the ClearSelection keyword in the command line, use one of these options:

- Type **ClearSelection**
- Type the shortcut **CS or Clears**

## Description

The ClearSelection keyword is a function keyword which is used to deselect the selected fixtures.

For information on the key see the [Clear Key](/grandma3/2-4/key_clear/).

For more information on the additional functions of the Clear keyword see [Clear](/grandma3/2-4/keyword_clear/), [ClearActive](/grandma3/2-4/keyword_clear_active/), [ClearAll](/grandma3/2-4/keyword_clear_all/).

## Syntax

ClearSelection
