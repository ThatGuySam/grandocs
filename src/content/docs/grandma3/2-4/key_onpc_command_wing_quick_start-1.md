### Das Gerät anschließen und einschalten

1. Gerät auspacken.
2. Verpackungs- und Polstermaterial entfernen. 
3. Gerät in einem geschlossenen Raum auf eine stabile Fläche stellen. 
4. grandMA3 onPC Software auf [www.malighting.com](https://help.malighting.com/grandMA3/2.4/HTML/www.%20malighting.com) herunterladen. 
5. Die Software auf PC oder Laptop installieren. Für Mindestanforderungen, siehe System Requirements grandMA3 im grandMA3 Online Benutzerhandbuch. 
6. Den grandMA3 onPC command wing mit dem PC oder Laptop mithilfe eines USB-Kabels verbinden.
7. Die grandMA3 Konsole oder grandMA3 onPC einschalten.
8. Den grandMA3 onPC command wing einschalten:\
   -IEC-Kaltgerätestecker mit der dafür vorgesehenen Buchse verbinden.  \
   -Netzstecker an die Steckdose anschließen. \
   -Netzschalter auf **I** stellen.  \
   -Netzschalter leuchtet rot. 

### Den grandMA3 onPC command wing ausschalten

Um den grandMA3 onPC command wing auszuschalten, Netzschalter auf **O** stellen.
