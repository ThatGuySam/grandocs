Timecode shows are arranged in a hierarchical structure, with **Track Groups** as children of the timecode show. Track groups can contain one or more **Tracks**.

**Markers** can be assigned to the track group. **Time Ranges** are assigned to tracks. Read the following topics to learn more about these elements.

To add a track group to a timecode show, follow these steps:

1. Edit an empty timecode pool object, for example, by using the swipey command. See [Pool Windows](/grandma3/2-4/wvm_pool/) for more information. This will open the timecode editor.
