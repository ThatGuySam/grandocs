### Inbetriebnahme

1. Gerät auspacken.
2. Verpackungsmaterial entfernen. 
3. Den grandMA3 viz-key mit einem USB-C-Kabel mit einem PC oder Laptop mit installierter grandMA3 onPC-Software verbinden.
4. Zum Ausschalten des grandMA3 viz-key die USB-Verbindung trennen.
5. Um die aktuelle Software herunterzuladen, gehen Sie auf [www.malighting.com](https://www.malighting.com/).\
   Um mehr Information zum Software-Update zu erhalten, lesen Sie im grandMA3 User Manual das Kapitel [Update grandMA3 viz-key](/grandma3/2-4/update_viz_key/).
