To enter the **/Selective **option keyword in the command line, use one of the options:

- Type **/Selective**
- Type the shortcut** /Se**

## Description

The /Selective option keyword is used to store or update selective data into presets regardless their mode. 

## Syntax

\[Function] Preset \["FeatureGroup\_Name" or FeatureGroup\_Number].\["Preset\_Name" or Preset\_Number] /Selective

## General Keywords

General keywords that use the /Selective option keyword:

- [CleanUp keyword](/grandma3/2-4/keyword_cleanup/)
- [Preset keyword](/grandma3/2-4/keyword_preset/)
- [Store keyword](/grandma3/2-4/keyword_store/)
- [Update keyword](/grandma3/2-4/keyword_update/)
