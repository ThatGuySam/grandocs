To enter the UserProfile keyword in the command line, use one of the options:

- Type **UserProfile** 
- Type **Userp**
- Type the shortcut **UPR**

## Description

The UserProfile keyword is an object keyword. It can be used to adjust settings in the UserProfile.

For more information on user profiles see [Create User](/grandma3/2-4/user_create/). For information on different settings see [User Settings](/grandma3/2-4/user_settings/).

## Syntax

\[Function] UserProfile \["UserProfile\_Name" or UserProfile\_Number] (Property \["Property\_Name"] \[Property\_Value])
