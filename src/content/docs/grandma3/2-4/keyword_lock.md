To enter the Lock keyword in the command line, use one of the options:

- Type **Lock**
- Type the shortcut **Loc**

## Description

The Lock keyword is a function keyword that locks objects. 

Objects that are locked are indicated by a padlock. 

For more information on locks and their types see [Lock and Unlock Pool Objects](/grandma3/2-4/wvm_pool_lock_unlock/).

## Syntax

Lock \[Object] \["Object\_Name" or Object\_Number]

## Option Keywords

The Lock keyword uses the following option keywords:

[/Recursive](/grandma3/2-4/ok_recursive/)
