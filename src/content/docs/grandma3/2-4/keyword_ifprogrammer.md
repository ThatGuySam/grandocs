To enter the IfProgrammer keyword in the command line, use one of the options:

- Press If If If
- Type **IfProgrammer**
- Type the shortcut **Ifp**

## Description

IfProgrammer is a function keyword that selects fixtures that contain values in the programmer.

|                                                  |                                                                                                          |
| ------------------------------------------------ | -------------------------------------------------------------------------------------------------------- |
| ![](/img/grandma3/2-4/important_gray-89eed8.png) | **Important:**                                                                                           |
|                                                  | Executed on its own, the IfProgrammer keyword only selects fixtures of the current command line default. |
