The **Sounds Pool** can contain sound files. The pool is part of the media pools.

|                                                  |                                                                                                                                                                                                    |
| ------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ![](/img/grandma3/2-4/important_gray-89eed8.png) | **Important:**                                                                                                                                                                                     |
|                                                  | The Sounds Pool has a limit of 100 MB for all objects. Files exceeding the limit will be highlighted with a red color in the import pop-up.The total size of all media pools is limited to 300 MB. |

Sound pool objects can be played back in the pool or added as a track in a timecode object.
