For more information, read the [Session topic](/grandma3/2-4/network_session/) in the [Networking section](/grandma3/2-4/network/).

## Change Name and Set IP Address

To change the name and to set the IP address of a grandMA3 Node, open the Network Interface Menu:

- Press Menu.\
  \- Opens the [menu select pop-up](/grandma3/2-4/wvm_menus/).
- Tap Network.\
  \- Opens the Network menu.
