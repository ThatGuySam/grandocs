To activate or deactivate a Quickey, tap on the particular pool object.

A virtual LED in the top right corner of each pool object indicates the current status of the Quickey, depending on the LED color:

- **Black**:**** Quickey is off. Some Quickeys are triggered without an LED color, for example, MENU or NUM1.

![](/img/grandma3/2-4/window_quickey_led4_v1-9_1-77024e.png)

- **White**: Quickey is active.

![](/img/grandma3/2-4/window_quickey_led3_v1-9-85598e.png)

- **Grey**: Certain Quickeys trigger keywords. The appropriate keyword is entered into the command line when the virtual LED is grey.

![](/img/grandma3/2-4/window_quickey_led_v1-9-48aca7.png)
