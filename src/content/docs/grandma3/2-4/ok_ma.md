To enter the **/MA **option keyword in the command line, use one of the options:

- Type **/MA**

## Description

The /MA option keyword is used to address a secondary function of an executor.

## Syntax

Assign \[Function] At Page \["Page\_Name" or Page\_Number].\["Executor\_Name" or Executor\_Number] /MA

Assign \[Function] At Executor \["Executor\_Name" or Executor\_Number] /MA

##

## Example

- To assign Goto to MA + executor of page 1, executor 201, type:
