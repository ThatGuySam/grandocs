Depending on the grandMA3 device type, there are up to three Ethernet ports on the rear panel. To set the IP addresses for the Ethernet ports, see [Interfaces and IP](/grandma3/2-4/network_interface/).

_grandMA3 _I/O Node DIN-Rail_ rear panel – _Ethernet
