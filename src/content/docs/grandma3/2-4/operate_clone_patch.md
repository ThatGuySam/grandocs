The cloning feature is a useful tool for duplicating existing fixtures and their data to new fixtures in the Patch menu for immediate use. The copy-and-paste functions automatically create a complete clone of the show data for the corresponding fixtures.

The following topics offer additional information:

- For general information about cloning, see [Clone Function](/grandma3/2-4/operate_clone/).
- To understand the Patch menu, see [Patch and Fixture Setup](/grandma3/2-4/patch/) and [Add Fixtures to the Show](/grandma3/2-4/patch_add_fixtures/).
- To position fixtures in the 3D viewer, see [Position Fixtures in the 3D Space](/grandma3/2-4/patch_position_fixtures/).

Take a look at the following example to better understand cloning fixtures.
