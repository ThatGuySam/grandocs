To enter the **/ForceUniversal **option keyword in the command line, use one of the options:

- Type **/ForceUniversal**
- Type the shortcut** /Forceu**

## Description

The /ForceUniversal option keyword adds universal data to a preset and sets preset mode to universal. Selective and global data will be removed in the preset.

## Syntax

\[Function] Preset \["FeatureGroup\_Name" or FeatureGroup\_Number].\["Preset\_Name" or Preset\_Number] /ForceUniversal

## General Keywords

General keywords that use the /ForceGlobal option keyword:

- [Preset keyword](/grandma3/2-4/keyword_preset/)
- [Store keyword](/grandma3/2-4/keyword_store/)
- [Update keyword](/grandma3/2-4/keyword_update/)
