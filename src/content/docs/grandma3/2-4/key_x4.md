Layout

Pressing and holding MA + X4 | Layout enters the Layout keyword into the command line.

|                                                                    |                            |
| ------------------------------------------------------------------ | -------------------------- |
| ![](/img/grandma3/2-4/icon_commandline-input_logo_v2-0-a59235.png) | User name\[Fixture]>Layout |
