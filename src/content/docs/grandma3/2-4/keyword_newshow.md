To enter the NewShow keyword in the command line, use one of the options:

- Type **NewShow **
- Type the shortcut **New**

## Description

The NewShow keyword is a function keyword that is used to create new shows.

## Syntax

****NewShow**** \["**Show\_Name"] (/Option)**

## Option Keywords

The NewShow keyword uses the following option keywords:

- [/All](/grandma3/2-4/ok_all/)
- [/DMXProtocols](/grandma3/2-4/ok_dmxprotocols/)
- [/LocalSettings](/grandma3/2-4/ok_localsettings/)
- [/NoShowData](/grandma3/2-4/ok_noshowdata/)
- [/OutputStations](/grandma3/2-4/ok_outputstations/)
- [/Path](/grandma3/2-4/ok_path/)
