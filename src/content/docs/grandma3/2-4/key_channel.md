##

## ID Types

Pressing Channel repeatedly switches through Channel, Generic, and ID types in use. ID types where no patched fixtures are assigned are not shown in the command line.

For more information, see the [Add Fixtures to the Show](/grandma3/2-4/patch_add_fixtures/).
