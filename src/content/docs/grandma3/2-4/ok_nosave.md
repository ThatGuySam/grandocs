To enter the **/NoSave **option keyword in the command line, use one of the options:

- Type **/NoSave**
- Type the shortcut **/NS** or **/Nosa**

## Description

The /NoSave option keyword defines for the show file not to be saved when loading a show, or shuttig down, restarting, or rebooting the console or your onPC. 

|                                                  |                                                                                                                                        |
| ------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| ![](/img/grandma3/2-4/important_gray-89eed8.png) | **Important:**                                                                                                                         |
|                                                  | The /NoSave option keyword only works in the conjunction with the [/NoConfirmation option keyword](/grandma3/2-4/ok_noconfirmation/).  |

## Syntax

**\[Function] (\[Object] \["Object\_Name" or Object\_Number]) /NoConfirmation /NoSave**

****

## General Keywords

General keywords that use the /NoSave option keyword:

- [LoadShow keyword](/grandma3/2-4/keyword_loadshow/)
- [Reboot keyword](/grandma3/2-4/keyword_reboot/)
- [Restart keyword](/grandma3/2-4/keyword_restart/)
- [ShutDown keyword](/grandma3/2-4/keyword_shutdown/)
