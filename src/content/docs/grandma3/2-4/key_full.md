Pressing Full enters the Full keyword into the command line.

Full sets the dimmer values to 100% in the selected fixtures.

|                                                                    |                          |
| ------------------------------------------------------------------ | ------------------------ |
| ![](/img/grandma3/2-4/icon_commandline-input_logo_v2-0-a59235.png) | User name\[Fixture]>Full |

For more information about Full, see the [Full keyword](/grandma3/2-4/keyword_full/).

## Location

Full is located in the command section.
