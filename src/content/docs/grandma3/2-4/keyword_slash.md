To enter the / \[Slash] in the command line, press /.

Description

The slash character has two functions:

- It is used to set options during a command.\
  For more information see the [Option Keywords](/grandma3/2-4/option_keywords/).
- It is used as a separator in the calculator.

|                                                  |                                                                                           |
| ------------------------------------------------ | ----------------------------------------------------------------------------------------- |
| ![](/img/grandma3/2-4/important_gray-89eed8.png) | **Important:**                                                                            |
|                                                  | When adding option keywords to your commands, make sure to use the slash without spaces.  |

## Syntax

\[Command] /Option \["Option\_Value"]
