#### Overview

Quickeys allow you to create soft versions of various hardkeys and functions.

#### Accessing Quickeys

You can access Quickeys by either:

- Using the Quickey Pool. Open the Add Window dialog, tap Data Pools, and then tap Quickeys. For more information about pools in general, see [Pool Windows](/grandma3/2-4/wvm_pool/).
- Using the [Quickey keyword](/grandma3/2-4/keyword_quickey/).
