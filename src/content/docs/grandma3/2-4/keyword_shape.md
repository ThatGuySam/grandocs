To enter the Shape keyword in the command line, use one of the options:

- Type **Shape**
- Type the shortcut **Sha**

## Description

The Shape keyword is an object keyword that is used to access shapes in the shape pool.

## Syntax

\[Function] Shape \["Shape\_Name" or Shape\_Number]

## Example

- To store shape 21, type:
