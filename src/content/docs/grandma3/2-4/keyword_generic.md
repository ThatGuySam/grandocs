To enter the **Generic **keyword in the command line, use one of the options:

- Press  Channel until Generic appears in the command line
- Type **F + 2**
- Type **Generic**
- Type the shortcut **Generi**

## Description

The Generic keyword is an object keyword wich is used to call fixtures of the ID type Generic in the programmer.

## Syntax

Generic\["Generic\_Name" or Generic\_Number]

## Example

- To select the fixture that uses the Generic ID 1, type:
