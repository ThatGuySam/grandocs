To get the Gel keyword in the command line press Preset Preset Preset .

Another way is to type **Gel** or the shortcut **Ge** into the command line. 

## Description 

The keyword gel provides the opportunity to edit or apply swatch book colors via command line. 

## Syntax

Gel "Swatch name"."Gel Name"

Gel "**Swatch name**"."Key"

Gel \[Swatch Id].\[Gel  Id]

**Gel \[Swatch Id]**

**Gel "****Swatch name****"**

##  

## Examples

\[Channel]> At Gel "Lee"."Mauve"

Sets the color of the selected fixtures to Lee´s color Mauve. 

 

\[Channel]> At Gel "Lee". "126"

Sets the color of the selected fixtures to Lee´s key 126 in swatch book. 

 

\[Channel]> At Gel 7.44

Sets the color of the selected fixtures to the 44th color of the seventh swatch book. 

 

\[Channel]> Export Gel 7

Exports the "Lee" swatch book.
