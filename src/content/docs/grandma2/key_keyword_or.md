To enter the keyword, type **Or** into the command line. 

## Description

Or is a helping keyword and is usually used along with the keyword If. 

For more information see [keyword If](/grandma2/key_keyword_if/). 

## Syntax

\[Function] \[Object-list] \[If] \[Object-list] Or \[Object-list]

##  

## Example

\[Channel]> Delete Cue 1 If Channel 2 or 3
