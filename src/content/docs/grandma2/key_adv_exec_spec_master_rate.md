It is possible to a assign 15 different Rate Masters to executors.

The Rate Masters can be assigned to sequences executors or effects. This makes it possible to sync multiple elements to the same rate.

Read more about assigning rate masters to sequences executors in the [Executor Options](/grandma2/key_adv_exec_options/) topic. Read about assigning a rate master to an effect in the [Use Predefined Effects topic](/grandma2/key_effects_predefined/). 

 

All the Rate Masters can be seen and adjusted in the [Masters window](/grandma2/key_adv_exec_master_window/).
