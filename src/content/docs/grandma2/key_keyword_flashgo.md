To execute the keyword FlashGo, type **FlashGo** or the shortcut **FlashG** into the command line. 

## Description

FlashGo executes the keyword Flash and jumps into the next cue.

For more information see the keyword [Flash](/grandma2/key_keyword_flash/). 

Syntax

 FlashGo \[Executor-list]

 

## Options

To get a list of all options of the keyword FlashGo, type the following into the command line:

\[Channel]> FlashGo /?

The keyword FlashGo has the same options as the keyword Goto. 

For further information see the keyword [Goto](/grandma2/key_keyword_goto/). 

 

## Example

\[Channel]> FlashGo Executor 1

Overwrites master level temporarily and jumps into the next cue.
