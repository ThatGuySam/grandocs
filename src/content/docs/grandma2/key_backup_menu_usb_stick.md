There are different things to keep in mind when using a USB stick.

Here is a list of good advice for the USB stick.

- Do not use a big stick. Do not go above 16Gb.
- Name the stick, but **do not** use a space in the name. Underscore or Dash are ok.
- It should be formatted in FAT32.
- If the stick have been in a Mac or PC make sure it has not been configured as something other than a "removable disk".

This is not rules in all software versions. But it is a list of advice that have been gathered by limitations in different software version.

So maybe not rules, but at least a good advice.
