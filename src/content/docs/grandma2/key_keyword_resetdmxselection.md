To go to the ResetDmxSelection keyword, type **ResetDmxSelection** in the command line or use the shortcut **ResetD**.

## Description

The ResetDmxSelection keyword clears the selection in the [DMX tester](/grandma2/key_patch_dmx_testing/).

ResetDmxSelection is a function keyword.

## Syntax

ResetDmxSelection

## Example

- Reset DMX Selection in the DMX tester.

\[Channel]> ResetDmxSelection
