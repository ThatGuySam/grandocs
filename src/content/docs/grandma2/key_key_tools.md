The key Tools is located in the command area on the left of the key Setup.

![](/img/grandma2/img_tools_key_3_2-f1a9c6.png)

_Location __Key __Tools _

 

Pressing Tools executes the [Tools keyword](/grandma2/key_keyword_tools/). The menu Tools opens.
