To go to the PreviewEdit keyword, press Prvw Prvw (= PrvwEdit).

You can also type **PreviewEdit** in the command line or use the shortcut **PreviewEd**.

To close the PreviewEdit, press Prvw Prvw again.

## Description

The PreviewEdit toggles between the live programmer and the blind preview editor.

The PreviewEdit keyword is an object keyword.

## Syntax

PreviewEdit

PreviewEdit On

PreviewEdit Off

## Example

Turn the preview editor on.

\[Channel]>PreviewEdit On

The preview encoder bar is displayed. The title bar of the fixture sheet, channel sheet, and layout view is red. The Prvw key is flashing.
