To get the Insert keyword in the command line press Move Move Move (= Insert).

You can also type **Insert** or the shortcut **Ins** in the command line.

## Description

The Insert keyword is a function keyword to insert pool objects between two other already taken pool objects. The following pool objects will be moved to the next empty destination.

When the Insert keyword is in the command line by pressing Move Move Move, you can insert pool objects by drag and drop.

## Syntax

**Insert** \[Object-list] **At** \[Destination Start-ID]

## Example

- To insert Group 5 between Group 8 and 9, type in the command line:

\[Channel]> Insert Group 5 At 9
