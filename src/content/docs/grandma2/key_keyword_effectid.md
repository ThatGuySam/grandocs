To get the EffectID keyword in the command line type **EffectID** or the shortcut **EffectI** in the command line:

## Description

EffectID is a helping keyword to indicate the EffectID layer.

The EffectID value cannot be adjusted the same way as other effectlayers. This value is given by calling an Effect.

For more information on effects see [effects](/grandma2/key_effects/).

## Syntax

EffectID

## Examples

- To switch the encoder to display the EffectID layer type in the command line:

\[Channel]> EffectID

 

- To knock out any EffectID values (effect reference) of the programmer for the selection/feature type in the command line:

\[Channel]> Off EffectID
