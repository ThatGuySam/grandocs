To go to the ViewPage keyword, press View View View (=ViewPage).\
The commandline shortcut is **VP**.

## Description

The ViewPage keyword calls the view pages. There are 11 view pages available.\
This is the same as press and hold the ![ma](/img/grandma2/ma_1-290dfd.png) and a V-Key.

The ViewPage keyword is an object keyword.

## Syntax

ViewPage \[ID]

## Example

Call view page 2.

\[Channel]> ViewPage 2
