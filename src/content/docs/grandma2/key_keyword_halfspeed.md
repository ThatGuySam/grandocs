To get the HalfSpeed keyword in the command line type **HalfSpeed** or the shortcut **Ha** in the command line.

## Description

HalfSpeed is a function keyword to divide the current speed by 2.

## Syntax

HalfSpeed \[Object-list]

## Example

- To halves the speed of executor 5 type in the command line:

\[Channel]> HalfSpeed Executor 5
