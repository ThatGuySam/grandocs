The key Go + (small) is located in the command area on the right of the key Pause.

![](/img/grandma2/img_goplus_small_key_3_2-424599.png)

_Location k__ey __Go + (small)_

 

- Pressing Go + enters the [Go keyword](/grandma2/key_keyword_go/) into the command line.

\[Channel]> Go

 

- Pressing Go + Go + enters the [Unpark keyword](/grandma2/key_keyword_unpark/) into the command line.

\[Channel]> Unpark
