**Blind** is a function that suppresses the output of the live programmer. Programming is possible without a live output. Switching off the blind mode brings back the live programmer including changes made during blind mode.

To execute this keyword press the key Blind, or type **Blind** or the shortcut **Bli** into the command line.

## Syntax

Blind

Blind On

Blind Off

## Description

Blind is a toggle function. This means that entering Blind without any helping keyword will toggle Blind mode on/off.

## Example

\[Channel]> Blind On

Turns on Blind mode (turns off output of programmer).
