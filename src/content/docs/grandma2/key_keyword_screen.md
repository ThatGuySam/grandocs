To get the Screen keyword in the command line, press ![ma](/img/grandma2/ma_1-290dfd.png) + View (= Screen).

You can also type **Screen** or the shortcut **Scr** in the command line .

## Description

The Screen keyword represents the screens.

The Screen keyword is an object keyword.

## Syntax

Screen \[screen-number]

Screen \[screen-number].\[window-number]

## Example

- Clear screen 2 to get an empty screen.

\[Channel]> Delete Screen 2

**Hint:**\
Another way to clear the screen is, press and hold the ![encoder](/img/grandma2/encoder_1-08ac5a.png) key and tap Clear Screen.

 

- Assigns mask 5 to the third window of the second screen.

\[Channel]> Assign Mask 5 Screen 2.3
