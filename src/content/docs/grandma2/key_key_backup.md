The key Backup is located on the right of the key Setup.

![](/img/grandma2/img_backup_key_3_2-66648e.png)

_Location k__ey __Backup _

 

Pressing Backup executes [Backup keyword](/grandma2/key_keyword_backup/) and opens the [Backup Menu](/grandma2/key_backup_menu/).

Pressing Backup Backup executes [SaveShow keyword](/grandma2/key_keyword_saveshow/) and saves the show data on the currently selected drive.
