The key Next is located in the command area on the right of the key Set.

![](/img/grandma2/img_next-key_v3-2-d2e512.png)

_Location k__ey __Next _

 

- If a pop-up dialog is open, for example the pop-up Choose store method, you can set the focus on the following options by pressing the key Next.\
   
- Pressing Next executes the [Next keyword](/grandma2/key_keyword_next/). The fixture with the lowest fixture ID is selected.

\[time] : Executing : Next

- Pressing and holding ![ma](/img/grandma2/ma_1-290dfd.png) + Next executes the [NextRow keyword](/grandma2/key_keyword_nextrow/). For more information see [MAtricks](/grandma2/key_matricks/).

\[time] : Executing : NextRow

- Pressing Next + Prev executes the [All keyword](/grandma2/key_keyword_all/).

\[time] : Executing : All
