To execute this keyword, type **BlackScreen** or the shortcut **BlackS** into the command line. 

## Description

The keyword BlackScreen is used to make single or all screens of the console turn black.

It is possible to revert the command for single and all screens of the console:

- Tap either Show Display or Show All Displays in the lower right corner of the screen. 

## Syntax

BlackScreen

 

## Options

To get a list of all options of the keyword BlackScreen, type the following into the command line:

\[Channel]> BlackScreen /?

The keyword BlackScreen has the following options:

| Option | Shortcut    | Description               |
| ------ | ----------- | ------------------------- |
| On     | No shortcut | All screens turn black.   |
| Off    | No shortcut | All screens turn back on. |

 

## Examples

\[Channel]> BlackScreen

All screens of the console turn black.

 

\[Channel]> BlackScreen Off 

All screens of the console turn back on.  

 

\[Channel]> BlackScreen 1

Screen 1 turns black.
