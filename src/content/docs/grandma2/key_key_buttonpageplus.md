The key Bt Pg + is located on the left of the 100 mm fader.

![](/img/grandma2/img_button_page_plus_key_3_2-b58021.png)

_Location key Bt__ Pg + _

 

- Pressing Bt Pg + executes the [ButtonPage command](/grandma2/key_keyword_buttonpage/). The button page switches to the next page.
