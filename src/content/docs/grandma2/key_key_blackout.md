The key B.O. is located in the command area on the right of the key Align.

![](/img/grandma2/img_bo-key_v3-2-800405.png)

_Location k__ey __B.O. _

 

As long as you press and hold B.O., the blackout mode is on. For more information see [Blackout keyword](/grandma2/key_keyword_blackout/).

The command line stays unaffected.

To disable the B.O. key see [User Settings](/grandma2/key_single_multi_systems_user_settings/).
