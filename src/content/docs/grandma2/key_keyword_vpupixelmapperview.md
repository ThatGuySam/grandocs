To access the **VPU Pixel Mapper View**, tap on an empty part of a screen. This must be at least 4 columns wide and 4 rows high. Tap Other and then VPU Pixel Mapper View.

The VPU pixel mapper view is described in the manual for the VPU. For more information about pixelmapping, see the [Pixelmapper](https://help.malighting.com/grandMA2/en/help/ma_vpu/key_pixelmapper.html) topic.

For more information about the view, see the [Console - VPU Pixel Mapper View](https://help.malighting.com/grandMA2/en/help/ma_vpu/key_console-vpu-pixelmapper-view.html)
