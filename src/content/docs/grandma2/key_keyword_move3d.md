To go to the Move3D keyword, type **Move3D** in the command line or use the shortcut **Move3**.

## Description

The Move3D keyword moves 3D objects in the [Stage View](/grandma2/key_patch_stage/).

If the Move3D keyword is used without values, the selected fixtures will be reset to zero.

If the option relative is used, the move function uses the relative position.

Move3D is a function keyword.

## Syntax

Move3D At \[x y z] /relative

## Example

Move the selected 3D object in the [Stage View](/grandma2/key_patch_stage/).

\[Channel]> Move3D At 2.5 -5 3.3
