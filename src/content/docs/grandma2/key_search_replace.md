The two functions **Search** and **Replace** are two very powerful tools in the grandMA2.

**Search** can be used to find a where different elements are used.

**Replace** is also doing a search, but it then uses the search result to replace one element with another. This can also be ranges of elements.

 

There is no graphic interface for these two functions. It is all done by typing in the command line entry.

There are some nice predefined macros for many of the search and replace functions - please read more about loading predefined macros in the [Import Predefined Objects topic](/grandma2/key_ei_import_predefined/).
