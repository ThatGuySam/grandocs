To go to the Tools keyword, type **Tools** in the command line or use the short cut **To**.

## Description

The Tools keyword opens or closes the tools menu.

![](/img/grandma2/menu_tools_3-4-bbcd39.png)

_Tools menu_

The Tools keyword is a function keyword.

**Hint:**\
Another way to open the Tools menu is, press the Tools key.

## Syntax

Tools

## Example

Open the tools menu.

\[Channel]> Tools
