To get the ShuffleSelection keyword in the command line, type **ShuffleSelection** or the shortcut **Shuf** in the command line .

## Description

The ShuffleSelection keyword mix-up the order of the selected fixture or channel.

**Hint:**\
Another ways to shuffle the selection is,\
\- go to the Tools menu and tap a Shuffle Selection Order\
or\
\- use the predefined macro ShuffleSelection.

## Syntax

ShuffleSelection

## Example

Mix-up the order of the selected fixtures or channels.

\[Channel]> ShuffleSelection
