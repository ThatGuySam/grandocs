The key Temp is located in the command area on the left of the key Top.

![](/img/grandma2/img_temp_key_3_2-16a41f.png)

_Location Key Temp__ _

 

- Pressing Temp enters the [Temp keyword](/grandma2/key_keyword_temp/) into the command line.

\[Channel]> Temp

- Pressing Temp Temp enters the [Toggle keyword](/grandma2/key_keyword_toggle/) into the command line.

\[Channel]> Toggle
