To get the SyncEffects keyword in the command line, press and hold ![ma](/img/grandma2/ma_1-290dfd.png) + Effects (= SyncEffects).

You can also type **SyncEffects** in the command line or use the shortcut **Sy**.

## Description

The SyncEffects keyword starts all running effects synchronous.

**Hint:**\
You can also use the Sync Effects button in the **Special Dialog I Effect Value**.

## Syntax

SyncEffects

## Example

Synchronize the PWM (= Pulse-width modulation) effect for the selected fixtures.

\[Channel]> SyncEffects
