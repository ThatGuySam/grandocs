+ \[Plus] is a helping keyword with various functions.

To execute the keyword **+ \[Plus]**, type **+** in the command line. 

## Syntax:

+ \[Plus] is used to combine multiple objects in a list or to indicate a relative value. When used as a relative indicator, with no value given, the value of 1 will be used.

If used as a starting keyword, + creates a selection list, which will be added to the current selection.

 

## Examples:

\[Channel]> Delete Cue 1 + 2

 Deletes cue 1 and 2 from the selected executor.

\[Channel]> At + 5

 Adds 5 % to the current dimmer value.

\[Channel]> + 5 Thru 7

 Adds channel 5, 6 and 7 to the current selection.

\[Channel]> Page +

 Same as Page + 1, which calls the next page.

 

**Hint:**\
To execute this function, you can also use the key + on the console. 

 

For more information see [+](/grandma2/key_key_plus/).
