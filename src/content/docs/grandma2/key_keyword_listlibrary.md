To get the ListLibrary keyword in the command line, type **ListLibrary** or the shortcut **ListL** in the command line.

## Description

The ListLirbrary keyword is a function keyword to display the fixture type library in the command line feedback window.

## Syntax

ListLibrary \[filter]

## Example

- To get all fixture types from the fixture type library displayed in the command line feedback window starting with **Martin**, type in the command line:

\[Channel]> ListLibrary "Martin\*"
