**ClearSelection** is a function used to deselect any selected fixtures.

To execute this keyword type **ClearSelection** or the shortcut **ClearS** into the command line. 

## Syntax

ClearSelection

The ClearSelection function clears the selection (deselects all fixtures).

To clear the selection press the button Clear once. 

For more information on further functions of the keyword [Clear](/grandma2/key_keyword_clear/) see [ClearActive](/grandma2/key_keyword_clearactive/) and [ClearAll](/grandma2/key_keyword_clearall/).
