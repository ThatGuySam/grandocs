To get the Fix keyword in the command line press Fix.

You can also type **Fix** directly in the command line.

## Description

Fix is a function keyword to fix objects on a page.

Fix is a toggle function. This means that using Fix without any helping keyword toggles the fixing of the objects on and off.

## Syntax

Fix \[executor list]

Fix On \[executor list]

Fix Off \[executor list]

## Examples

- To fix executor 1 through 5 on the current page type in the command line:

\[Channel]> Fix On Executor 1 Thru 5

 

- To toggle executor 3 from changing page, or not to changing page, type in the command line:

\[Channel]> Fix Executor 3
