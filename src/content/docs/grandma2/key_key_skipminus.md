The key <<<  \[GoFastBack] is located in the command area on the left of the key Learn.

![](/img/grandma2/img_go-fast-back-key_v3-2-a88485.png)

_Location <<< \[GoFastBack]_

 

- Pressing <<< enters [<<< \[GoFastBack\] keyword](/grandma2/key_keyword_skipminus/) into the command line.

\[Channel]> <<<

- Pressing <<< <<< enters [Black keyword](/grandma2/key_keyword_black/) into the command line.

\[Channel]> Black

- Pressing <<< <<< <<< enters [ToZero keyword](/grandma2/key_keyword_tozero/) into the command line.

\[Channel]> ToZero
