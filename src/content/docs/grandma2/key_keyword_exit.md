To enter the keyword Exit into the console, type **Exit** or the shortcut **Exi** into the command line. 

## Description 

Exit disconnects the connection to Telnet. 

For more information see the keyword [Telnet](/grandma2/key_keyword_telnet/). 

## Syntax

Exit

 

## Example

\[Channel]> Exit

Disconnects Telnet.
