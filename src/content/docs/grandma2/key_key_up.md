The key Up is located in the command area above the key Set.

![](/img/grandma2/img_up_key_3_2-c325be.png)

_Location __Key __Up _

 

- If you press the key Up, the scroll bar of the window or pop-up, in which the focus is set, moves up.
- If you press and hold ![ma](/img/grandma2/ma_1-290dfd.png) + Up, the monitor wing of the console moves up.
