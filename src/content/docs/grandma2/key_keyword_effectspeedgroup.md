To enter the keyword EffectSpeedGroup into the console, type **EffectSpeedGroup** or the shortcut **EffectSp** into the command line. 

## Description

EffectSpeedGroup calls the effect layer **Speed Group**.

For more information on effect layers see [Layers in sheets – Channel sheet and fixture sheet](/grandma2/key_of_layers_in_sheets_channel_and_fixture/). 

To change to different layers, press ![ma](/img/grandma2/ma_1-290dfd.png) + Time. 

## Syntax

EffectSpeedGroup

 

## ​Example

\[Channel]> EffectSpeedGroup

Jumps to the effect layer Speed Group.
