To get the FadePath keyword in the command line type **FadePath** or the shortcut **FadeP** in the command line.

## Description

FadePath is an object type representing the possible transition paths of a cue part.

## Syntax

FadePath \[ID]

## Example

To assign a high-low fadepath to the main part of cue 4 type in the command line:

\[Channel]> Assign FadePath 7 At Cue 4 Part 0
