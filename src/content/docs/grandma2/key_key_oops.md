The key Oops is located in the command area on right of the key Edit.

![](/img/grandma2/img_oops-key_v3_2-2c6620.png)

_Location k__ey __Oops _

 

- Pressing Oops deletes the last keyword or number you entered in the command line.\
   
- If the command line is empty, pressing Oops executes the [Oops keyword](/grandma2/key_keyword_oops/).\
  The pop-up Confirm Undo opens.

 

![](/img/grandma2/popup_confirm-undo_v3-2-96d397.png)

_Pop-up Confirm Undo_

 

- Pressing and holding the key Oops for about 2 seconds, opens the pop-up Multi-Oops.

![](/img/grandma2/popup_multi-oops_v3-2-fd557c.png)
