The key Select is located in the command area on the left of the key Off.

![](/img/grandma2/img_select_key_3_2-8650b1.png)

_Location __Key __Select_

 

- Pressing Select enters the [Select keyword](/grandma2/key_keyword_select/) into the command line.

\[Channel]> Select

- Pressing Select Select enters the [SelFix keyword](/grandma2/key_keyword_selfix/) into the command line.

\[Channel]> SelFix
