To get the AllChaseExecutors keyword in the command line type **AllChaseExecutors** or the shortcut **AllC** in the command line.

## Description

AllChaseExecutors is used as an alias to address all executors with chasers assigned.

## Syntax

\[Function] AllChaseExecutors

## Example

- To turn off all executors with chasers type in the command line:

\[Channel]> Off AllChaseExecutors
