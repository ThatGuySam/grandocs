To execute the keyword Mask press and hold 

![ma](/img/grandma2/ma_1-290dfd.png) and Group Group Group or type **Mask** into the command line.

## Description

The Mask keyword is an object keyword to access Masks in the Mask pool. The default function is [Select](/grandma2/key_keyword_select/).

For more information, see [worlds, filters, and masks](/grandma2/key_wfm_world_filter_mask/).

## Syntax

**\[Function] Mask** \[ID]

## Examples:

 

\[Channel]> Mask 3

 

Selects mask 3 in the mask pool.

 

\[Channel]> Mask "My Mask"

 

Selects mask with the name "My Mask".
