## Title Bar Icons

The title bar provides you with information icons and icons with a function.

The following table explains what the icons in title bar mean.

| Icon                                                  | Meaning                                                                                                                                                           |
| ----------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| ![](/img/grandma2/icon_ghost_v3-2-88ba6d.png)         | A mask is in use. For more information, see [Use Masks in the Sheets](/grandma2/key_wfm_use_masks/).                                                              |
| ![](/img/grandma2/icon_ghost-red_v3-2-e08051.png)     | A mask is assigned. For more information, see [Use Masks in the Sheets](/grandma2/key_wfm_use_masks/).                                                            |
| ![](/img/grandma2/icon_home_v3-2-f54f1b.png)          | Go home to the Setup start screen.                                                                                                                                |
| ![](/img/grandma2/icon_change-screen_v3-2-221a7a.png) | Opens the change screen pop-up.                                                                                                                                   |
| ![](/img/grandma2/icon_close_v3-2-8bb8c2.png)         | Deletes the window or closes pop-ups.                                                                                                                             |

## Command Line and Message Center Icons

The [command line](/grandma2/key_widget_command_line/) and the message center provides you with information about the:

- Network
- Show
- Power
- Chat
- CITP

Some icons can be disabled or enabled in the message center options. For more information, see [Message Center](/grandma2/key_message_center/).
