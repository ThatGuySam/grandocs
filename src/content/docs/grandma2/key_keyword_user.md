To use to the User keyword, type **User** the shortcut **Us** in the command line**.**

## Description

The User keyword displays user details in the  command line feedback window.

- Name
- Password
- UserProfile
- Rights
- LoggedIn
- Info

![](/img/grandma2/window_commandline-feedback_user_v3-2-262cff.png)

_Command Line Feedback List User_

 

The User keyword is an object keyword.

For more information, see [create user profiles and user](/grandma2/key_single_multi_systems_create/).

## Syntax

User \[ID]

## Example

- List user details of user 1.

\[Channel]> List User 1

 

- List all user details.

\[Channel]> List User
