To get the ListShows keyword in the command line, type **ListShows** or the shortcut **ListS** in the command line.

## Description

The ListShows keyword is a function keyword to display the show files on the selected drive in the **Command Line Feedback** window.

## Syntax

ListShows \[Filter]

## Examples

- To get all show files in the command line feedback window displayed, type in the command line:

\[Channel]> ListShows

 

- To get all show files beginning with f in the command line feedback window displayed, type in the command line:

\[Channel]> ListShows f\*
