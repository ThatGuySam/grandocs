The key User 1 is located on the right of screen 1.

![](/img/grandma2/img_user1_key_3_2-f9f257.png)

_Location __Key __User 1 _

 

If you press User 1, the [X-Keys](/grandma2/key_key_xkey/) and [X-Key labels](/grandma2/key_widget_x_key_labels/) on screen 1 change to the User 1 page.
