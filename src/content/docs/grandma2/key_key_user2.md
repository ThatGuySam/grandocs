![](/img/grandma2/img_user2_key_3_2-81b388.png)

_Location __Key __User 2 _

 

If you press User 2, the[ X-Keys](/grandma2/key_key_xkey/) and [X-Key labels](/grandma2/key_widget_x_key_labels/) change to the User 2 page on screen 1.
