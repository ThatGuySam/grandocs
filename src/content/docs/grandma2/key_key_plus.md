The key + is located next to the numeric keys in the command area.

![](/img/grandma2/img_plus_key_3_2-2504b9.png)

_Location k__ey __\[Plus] + _

 

- Pressing + enters [+ \[Plus\] keyword](/grandma2/key_keyword_plus/) into the command line.

\[Channel]> +

- Pressing + + adds 10 % of dimmer value to the selected fixture or channel.

Macro: At +10
