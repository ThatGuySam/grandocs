**Important:**\
The keyword Escape is only used within a macro. To cancel commands that have not been executed, press Esc. For more information see the [key Esc](/grandma2/key_key_escape/). 

To enter the keyword into the console, type **Escape** or the shortcut **Es** into the command line. 

## Description

The keyword Escape closes modal dialogs within a macro. 

## Syntax

Escape
