To open the options of the timer pool, tap the ![](/img/grandma2/icon_yellow-ball_3_2-329eb0.png) in the upper left corner of the pool.

The option dialog opens.

![](/img/grandma2/window_timer-pool-options_v3-3-542d8c.png)

_Setting the options of the timer pool_

1. To adjust the general settings such as the frame color, sheet style, font or the reset of colors, see [Adjust pool options](/grandma2/key_pools_options/).
2. To start or pause the timer directly in the pool, swipe the button Select to Start / Stop.\
   -To start the timer, tap the pool object once.\
   -To pause it, tap it once again. 

**Hint:**\
If you swipe the button back to Select, tapping the timer in the timer pool selects it.
