It is possible to delete single DMX channels in the table Attributes.

## Delete a single DMX channel

1. Select the DMX channel in the table Attributes and click **Delete **below the table Attributes.
2. A warning pop-up appears.

![](/img/utility/warning-pop-up_delete-dmx-channels_v1_3_1-88934b.png)

_Warning pop-up_

3. To delete the DMX channel, click **Delete**.\
   To abort the operation, click **Cancel**.  

The DMX channel is now deleted.

**Important:**\
The DMX addresses Fine and Coarse may have to be adjusted for the following attributes in some cases.
