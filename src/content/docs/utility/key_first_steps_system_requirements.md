If you want to run the MA Fixture Builder on your PC, here is what it takes.

 

| Operating system | Minimum                     |
| ---------------- | --------------------------- |
|                  | Microsoft Windows® 7        |
|                  | Microsoft Windows® 8        |
|                  | Microsoft Windows® 8.1      |
|                  | Microsoft Windows® 10       |
|                  | Apple OS X 10.10.5 Yosemite |

Additional requirements to use certain features:

- To use the online help and download the latest version of MA Fixture Builder you need internet access.
- To save on an USB stick you need an USB 2.0 or 3.0 port.

We recommend you visit your PC manufacturer´s website for information on updated drivers and hardware compatibility.
