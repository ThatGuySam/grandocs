It is possible to [export the fixtures created](/utility/key_export_fixture/) from the MA Fixture Builder as an .xml file and save it to a USB drive or a folder.

This .xml file can be imported into the console or the onPC software.

You can also [import ](/utility/key_import_fixture/).xml files from the subfolder "library" of dot2 and gma2. 

This may be useful if you would like to adjust fixtures or adjust an already existing one.
