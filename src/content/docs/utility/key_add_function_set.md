It is possible to add function sets:

1. Click **Add** below the table Function Sets. 
2. The window Add Function opens. 

![](/img/utility/add_function_set_v1_3_1-26e98a.png)

_Add a function set_

3. Enter the function name in the corresponding input field. 
4. Enter the DMX range in the input fields Dmx From and Dmx To.
5. Select the slot in the drop-down list.
6. To save the function set, click Save. 
7. To abort the operation, click Cancel. 
8. The settings are now saved.
