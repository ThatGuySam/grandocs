# dot2 landmark pages

Routes are `/dot2/<slug>/`; raw markdown is `/dot2/<slug>.md`. No version segment.
Cite the page's `sourceUrl` (`help.malighting.com/dot2/...`).

## Commands & command line

| Topic                       | Slug                       |
|-----------------------------|----------------------------|
| Commands overview (index)   | `key_commands_overview`    |
| Command line view           | `key_view_commandline`     |

| Glossary                    | `key_glossary`             |
| Calculator view             | `key_view_calculator`      |
| Oops view                   | `key_view_oops`            |

dot2 documents commands as a single overview list (`key_commands_overview`) rather
than one page per keyword. On the console itself: press **Help**, the command key,
then **Please**. Quote any syntax shown verbatim — never invent it.
